# 🏃 Модуль фитнес-трекера

Программный модуль для расчёта шагов, дистанции, скорости и потраченных калорий в зависимости от типа активности.

## 📦 Состав проекта

- **daysteps** — учёт дневной активности (шаги, дистанция, калории при ходьбе).
- **spentcalories** — расчёт калорий для бега и ходьбы с учётом роста, веса и продолжительности.

## 🛠️ Стек

- Golang
- Стандартная библиотека (`time`, `strconv`, `strings`, `log`)

## 🚀 Запуск

```bash
go run main.go
